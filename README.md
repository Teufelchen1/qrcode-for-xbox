# qrcode-for-xbox
Show a QR Code symbol on your TV using your Xbox

Setup and install nxdk. See https://github.com/XboxDev/nxdk.
Then run `make`. 

If you are on macOS and want to run it locally:
`make macOS && ./main.out`
